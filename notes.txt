## Requirement List
1. HTML page with a button to change the background color.
2. CSS to style the button and the page.
3. JavaScript to handle the button click event and change the background color.

## Pseudocode
1. Create an HTML file with a button element.
2. Style the button using CSS.
3. Add a JavaScript file and link it to the HTML file.
4. In the JavaScript file, add an event listener to the button.
5. When the button is clicked, generate a random color.
6. Change the background color of the body to the generated color.
